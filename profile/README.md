# NEMONEMO

소규모 팀과 사이드 프로젝트를 위한  
**팀 일정 · 역할 기반 투두 · 진행 상황 시각화 플랫폼**입니다.

---

## 🧠 문제 정의

소규모 팀은 보통  
카카오톡, 디스코드, 노션, 캘린더 등 여러 도구를 동시에 사용합니다.

그 결과,
- 일정이 채팅에 묻히고
- 마감 관리가 느슨해지며
- 팀원별 진행 상황을 한눈에 보기 어렵습니다.

---

## 🎯 프로젝트 목표

- 팀 단위 일정과 투두를 **하나의 구조화된 흐름**으로 관리
- 역할 기반 작업 분담과 진행 상황을 **직관적으로 시각화**
- 사이드 프로젝트 팀이 **가장 먼저 떠올리는** 일정 관리 도구
